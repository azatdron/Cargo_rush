Cargo Rush v0.1

Первый прототип Telegram Mini App / mobile web game.
Открой index.html в браузере.

Функции:
- главный экран без emoji
- мультяшная машина
- кнопка ПОЕХАЛИ
- таймер 30 минут
- прогресс-бар
- начисление Cargo Coin
- лидеры недели
- автосалон
- гараж
- рефералы
- сохранение прогресса в localStorage
